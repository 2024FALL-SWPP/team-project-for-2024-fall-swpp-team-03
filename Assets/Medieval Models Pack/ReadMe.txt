The 'Resources' directory contains the files necessary for the correct display of objects.
To avoid problems, we recommend you refrain from editing these files.

Working objects are located in the 'Prefabs' directory in the form of Prefabs.

In the directory 'Scenes' you can find two demo-scenes. They have the same functionality, but use different render pipelines.

Also this package contains one fire shader. You can find a simple pdf-instruction in the 'VFX/CampFire' directory

If you have any problems with this package, at first we recommend you to read file 'Switching render pipelines.pdf'. After that if you still have any questions, you can contact us by email - airyng@yandex.ru
